# House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using Linear Regression on the California Housing dataset.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Results
- MAE: 0.533
- RMSE: 0.746
- R² Score: 0.576
